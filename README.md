# Acceso a datos en Java

Proyecto realizado en Java que trabaja con acceso a bases de datos mediante JDBC.

## Tecnologías utilizadas
- Java
- JDBC
- MySQL / Bases de datos relacionales

## Descripción
Este proyecto fue desarrollado como práctica del ciclo formativo **Desarrollo de Aplicaciones Multiplataforma (DAM)**.  
La aplicación se conecta a una base de datos y permite realizar operaciones de consulta y manipulación de datos desde Java.

## Contenido del repositorio
- Ejercicios de acceso a bases de datos en Java
- Conexión a base de datos
- Consultas y manejo de resultados
